FreeDream README

Author: Caroline Monmerqué
Author URI: http://dreamgratuit.canalblog.com


FreeDream is a clean and simple theme using HTML5 and CSS3 properties.

Feel free to use as you please. I would be very pleased if you could keep the 
Dreamweaver Gratuit link in the footer. Thanks and enjoy.


Basic Setup
---------------------------------

- Archives Page
Setup your Archives page by creating a new page in the Wordpress admin and 
selecting the Archives template.


- Footer Links
The footer links can be edited in the footer.php template file.
